# BOOST

BOOST is the Flutter client for `https://new.moneyfly.top`.

- API: `https://new.moneyfly.top/api/v1`
- Releases: `https://github.com/moneyfly004/BOOST/releases`
- Source: `https://github.com/moneyfly004/BOOST`

## Version

The BOOST client version starts at `1.0.0`.

## Build

Builds are produced by GitHub Actions only. Push to `main` or run the
`BOOST Build` workflow manually in GitHub Actions.
